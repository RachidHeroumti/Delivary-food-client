# Delivary-food

## Description
The client-side of our delivery food website is crafted using React.js and Tailwind CSS.
Enjoy a user-friendly interface with responsive design, intuitive navigation, 
and seamless integration for ordering your favorite meals online.

- built with React JS 
- User-friendly interface  

## Getting Started

1. **Prerequisites:**
    - Node.js and npm (or yarn) installed on your system.

2. **Installation:**
    - Clone the repository: `git clone https://github.com/RachidHeroumti/Delivary-food-client`
    - Navigate to the project directory: ./
    - Install dependencies: `npm install`

3. **Running the App:**
    - Start the development client: `npm run dev`
